# CONFLUENCE.md — Oracle

## Critério de confluência com indicadores técnicos

Conte quantos dos 6 indicadores confirmam a direção:

### Checklist de Confluência LONG
1. EMA9 > EMA21 → ✅ trend bullish
2. MACD em BULL ou histograma expanding → ✅ momentum
3. RSI entre 35-65 → ✅ zona saudável
4. OBV crescente → ✅ volume confirma
5. BB% < 0.5 → ✅ espaço para subir
6. ATR estável/crescente → ✅ mercado ativo

**Resultado:**
- 5-6 confluem → confiança 8-10, sinal forte
- 4 confluem → confiança 6.5-7.9, sinal moderado
- 3 confluem → confiança 5-6.4, hold
- < 3 confluem → hold obrigatório

### Contexto macro amplifica/reduz
- Variação BTC > +1% + OBV crescente → +1 ponto de confiança
- Headlines de crash, hack, regulação → -2 pontos
- Dominância BTC > 55% e subindo → cautela, reduzir confiança

## Regra de Paciência
Prefira hold a um sinal com confiança < 6.5. Um não-trade é uma decisão válida.
