# RSI_STRATEGY.md — Hype Beast

## Interpretação de RSI com dados de proxy
Quando não há dados diretos de HYPE, use BTC como proxy:
- BTC variação 24h < -3% + volume alto → RSI provavelmente oversold → oportunidade de compra
- BTC variação 24h > +3% + volume alto → RSI provavelmente overbought → aguardar retração
- BTC dominance caindo → altcoins como HYPE tendem a outperformar

## Regras de confiança
- Sinal forte (confiança 0.8+): múltiplos indicadores alinhados + notícias neutras/positivas
- Sinal médio (0.6-0.79): maioria alinhada, 1 incerto
- Sinal fraco (< 0.6): retornar hold
