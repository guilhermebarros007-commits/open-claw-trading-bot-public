# RSI_STRATEGY.md — Hype Beast

## Interpretação de indicadores técnicos para SOL

### RSI(14)
- RSI < 30 → oversold extremo → oportunidade de compra forte
- RSI 30-40 → oversold → oportunidade se confirmado por MACD/OBV
- RSI 40-60 → neutro → aguardar segundo sinal
- RSI > 65 → momentum enfraquecendo → considerar saída
- RSI > 75 → overbought → fade position

### MACD
- Histograma expandindo positivo → momentum crescente → confirma compra
- BULL crossover → trend virando → entrada se RSI < 45
- BEAR crossover → trend virando negativa → saída ou short

### Bollinger Bands + ATR
- BB% < 0.2 + ATR alto → squeeze rompendo para baixo → potencial reversão
- BB% > 0.8 + ATR alto → tendência forte, pode continuar
- ATR baixo → mercado sem volatilidade → sinais menos confiáveis

### OBV
- OBV crescente + preço caindo → divergência bullish → setup de compra
- OBV caindo + preço subindo → divergência bearish → cautela

## Regras de confiança (escala 0-10)
- 8.0+: ≥ 4 indicadores alinhados + contexto macro favorável
- 6.5-7.9: 3 indicadores alinhados, 1-2 incertos
- < 6.5: retornar hold obrigatório
